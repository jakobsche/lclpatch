FormAdjust

adjusts a form on the screen

Declaration

Source position: formex.pas line 13

procedure FormAdjust(

  Form: TForm

);

Arguments

Form

  

form that has to be placed on the screen

Description

Not all operating systems support TForm.Position = poDefault. If you call this procedure in a form's event handler, for example OnCreate, you can adjust it similarly to Position = poDefault under Windows SE. Call this procedure with the form instance as parameter to make your form more independent on system supported functionality.
